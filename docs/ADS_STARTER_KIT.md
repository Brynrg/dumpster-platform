# Ads Starter Kit

Campaign structure, keywords, ad copy, and conversion events for regional dumpster and dump trailer ads.

---

## Campaign Structure by Region

| Region | Campaign focus |
|--------|----------------|
| TX (Spring / North Houston) | Dump trailers + dumpsters |
| FL (Brevard County) | Dump trailers + dumpsters |

Create separate campaigns or ad groups for:
- **Dump trailers** (rental, delivery)
- **Dumpsters** (roll-off, rental)

---

## Keyword Themes

**Dump trailers:**
- dump trailer rental [city]
- dump trailer [city]
- small dump trailer rental
- dump trailer delivery [region]

**Dumpsters:**
- dumpster rental [city]
- roll off dumpster [city]
- dumpster rental near me
- construction dumpster [region]

**Negative keywords:**
- free
- cheap
- diy
- plans
- for sale (when targeting rental)
- jobs
- salary

## Conversion Events

Track these events for optimization:

| Event | Description |
|-------|-------------|
| `start_check` | User begins availability check |
| `submit_request` | User submits rental/quote request |
| `notify_opt_in` | User opts in to SMS/email notifications |
| `lead_created` | Lead record created in system |
