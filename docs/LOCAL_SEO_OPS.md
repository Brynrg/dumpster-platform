# Local SEO Operations

This module is admin-only and focused on compliant local SEO execution.

## GBP Weekly Cadence Checklist

- Verify primary category and service area are current for each market.
- Add or refresh real project photos tied to actual work completed.
- Confirm hours, phone, and business name match current NAP standards.
- Review questions and answers for accuracy and policy-safe wording.
- Publish one policy-compliant update post for useful local context.

## Review Request Best Practices (Compliant)

- Ask only after a real interaction or completed service touchpoint.
- Never offer money, discounts, or incentives in exchange for reviews.
- Do not ask for only positive reviews or suppress negative feedback.
- Keep language neutral and optional (no pressure or guarantees).
- Store requests as drafts first and review for compliance before sending.

## Citation Priority Order

1. Google Business Profile consistency checks
2. Apple Business Connect
3. Bing Places
4. Yelp
5. Nextdoor
6. Chamber listings
7. Angi and Thumbtack

## How To Use the Admin Module

1. Open `/admin/seo` and select the target market.
2. Use **Tasks** to track cadence work (GBP, reviews, citations, content, maps).
3. Use **Reviews** to build compliant SMS/email draft copy only.
4. Use **Citations** to update listing URLs, NAP values, and verification dates.
5. Use **Expansion** to review top city slug demand and candidate city pages.
6. When adding new cities, update `src/lib/cities.ts` and run lint/build checks.
